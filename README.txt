encryption.exe takes 3 arguments to run. 
First argument is the file you would like encrypted.
Second argument is the output encrypted file.
Third argument is the passphrase to encrypt the file.

Ex.
encryption.exe fileToEncrypt.txt encryptedFile.txt passphrase

decryption.exe takes 3 arguments to run. 
First argument is the file you would like decrypted.
Second argument is the output decrypted file.
Third argument is the passphrase used to encrypt the file.

Ex. 
decryption.exe fileToDecrypt.txt decryptedFile.txt passphrase
